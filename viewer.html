Technical specifications regarding security and image rendering.

Our cloud-hosted OpenSlide engine supports gigapixel whole slide images (WSI) exceeding 40 Gigabytes. For extremely massive files, our secure uploader uses segmented AWS multipart uploads, preventing browser crashes and connection drops.

Absolutely. Our platform complies fully with HIPAA data security parameters. Uploaded slides are decoded entirely within memory tiles using sandboxed virtual machines—no metadata records or patient identity links are ever retained within the temporary S3 bucket.

The PCI Digital slide viewer is optimized for standard modern web browsers that support hardware WebGL rendering. This includes Google Chrome, Microsoft Edge, Safari, and Firefox. No desktop software downloads are required.
